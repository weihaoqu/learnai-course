# 5 Project Ideas for High School Students
## Zero coding background | 30-40 minutes each | Uses Claude Code or ChatGPT

---

## Project 1: My AI Study Buddy
**Pitch:** Build a personal AI tutor that quizzes you instead of giving answers
**Tool:** ChatGPT or Claude (free tier works)
**Time:** 30 min
**Artifact:** Exported study system + screenshot of tutoring session
**Course connection:** Course 4, Track A (Socratic Tutor, Deep Understanding)
**Difficulty:** Easiest — good for shy or cautious students

---

## Project 2: Design a Playable Game
**Pitch:** Design and build a working browser game with AI
**Tool:** Claude Code (in Terminal)
**Time:** 35-40 min
**Artifact:** A playable HTML game running in the browser
**Course connection:** Course 4, Track C (Game Design) + Course 1 (Claude Code)
**Difficulty:** Medium — most exciting, biggest "wow" factor with parents

---

## Project 3: 60-Second Explainer Script
**Pitch:** Write a complete short video script with scene descriptions
**Tool:** ChatGPT or Claude (free tier works)
**Time:** 25-30 min
**Artifact:** Polished script + scene-by-scene image descriptions
**Course connection:** Course 4, Track C (Video Plan)
**Difficulty:** Easy — good for creative/verbal students

---

## Project 4: AI Myth Buster
**Pitch:** Catch AI lying and write a fact-check report
**Tool:** ChatGPT or Claude + Google for verification
**Time:** 30 min
**Artifact:** "3 AI Myths Busted" report with evidence
**Course connection:** Course 4, Track B (Check Claims, AI Hype)
**Difficulty:** Medium — good for skeptical/analytical students

---

## Project 5: Story Bible + First Chapter
**Pitch:** World-build a story universe and co-write the opening with AI
**Tool:** ChatGPT or Claude (free tier works)
**Time:** 30-35 min
**Artifact:** Story bible + edited first chapter
**Course connection:** Course 4, Track C (Story Writing)
**Difficulty:** Easy — good for creative/imaginative students

---

## Matching Students to Projects

| Student type | Best project |
|---|---|
| Shy or cautious | Project 1 (Study Buddy) — low stakes, private |
| Wants to impress | Project 2 (Game) — biggest visual payoff |
| Creative writer | Project 5 (Story) — imagination-driven |
| Analytical thinker | Project 4 (Myth Buster) — detective work |
| Social/performer | Project 3 (Video Script) — presentation-ready |

## If a Student Finishes Early
- Improve their project with a more specific prompt
- Try a second project
- Explore the Course 4 quizzes
- Help another student (peer teaching)
