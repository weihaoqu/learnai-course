Subject: AI Training Session — Preparation Steps (Action Required)

Dear Parents,

Thank you for enrolling your child in the AI training session at Monmouth University (Room HH216, Summer 2026).

**Before the session, please complete these 3 steps with your child:**

**Step 1: Subscribe to Claude Pro**
- Go to https://claude.ai
- Click "Sign up" and create an account (use your email for minors)
- Subscribe to Claude Pro ($20/month) — required for the hands-on coding tool
- You can cancel anytime after the session

**Step 2: Install Claude Code on your child's laptop**
- Install Node.js from https://nodejs.org (download the LTS version)
- Open Terminal (Mac: search "Terminal") or Command Prompt (Windows: search "cmd")
- Type this command and press Enter:
  npm install -g @anthropic-ai/claude-code
- Then type: claude
- Follow the prompts to log in with the Claude account

**Step 3: Review the course website**
- Visit: https://weihaoqu.github.io/learnai-course/
- Read the consent form and study procedure on the page
- Review the "Prepare Before the Session" section

**On session day, your child should bring:**
- Their laptop with charger
- Claude Code working (type "claude" in Terminal to verify)

**If you need help with setup**, email me at wqu@monmouth.edu or call 732-263-5396. I'm happy to walk you through it.
