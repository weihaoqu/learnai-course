Graduate AI Engineering Track — CS/SE Graduate Students

A four-semester advanced course sequence built around Rohit Ghumare's MIT-licensed AI Engineering from Scratch curriculum: first-principles implementation, modern LLM systems, agent engineering, production infrastructure, safety, and capstone delivery.

Course Positioning

This track is designed for graduate CS and Software Engineering students who can read code, reason about systems, and sustain a semester-scale build. It converts a large self-paced curriculum into a teachable university sequence with measurable phase outcomes.

  • Audience: MS CS, MS Software Engineering, AI concentration, or senior honors bridge students.
  • Prerequisites: Data structures, algorithms, software engineering practice, Python proficiency, and working linear algebra readiness.
  • Pedagogy: Build from scratch first, then compare against modern frameworks and production patterns.
